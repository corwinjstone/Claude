# Sponsorship & Partnerships — AI Agent Constraints

Layers on top of `teams/00-design-system/ai-agents.md`.

## You MUST NOT

1. Insert sponsor logos, links, or copy inside the registration funnel
   between "start" and "confirmation."
2. Render sponsor-brand fonts (Coca-Cola Spencerian, etc.) anywhere in
   product code. Sponsor wordmarks render as images.
3. Substitute sponsor colors for RunClub design tokens. Sponsor accent
   is additive, never replacing.
4. Generate co-branded copy in the sponsor's marketing voice. Use the
   division's copy bank.
5. Generate code that exposes athlete PII, email addresses, or
   contact data to sponsor integrations.

## You MUST

1. Treat each sponsor accent color as a project token registered via
   the Design System team. Reference it by token name only.
2. Use single-color sponsor logos on RunClub surfaces; multi-color
   logos require a Design System team review.
3. For sanctioning-body logos (USATF, USAT, ITRA, etc.), follow the
   sanctioning body's usage guidelines — their guidelines override
   ours where they conflict.
